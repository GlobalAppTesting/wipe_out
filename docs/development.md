## Setup

Follow the steps below to setup wipe_out locally:

  * Make sure you're running on Ruby 3.0.0 or newer
  * sqlite is installed (required for tests)

```bash
git clone https://github.com/GlobalAppTesting/wipe-out
cd wipe-out
bundle install
```
